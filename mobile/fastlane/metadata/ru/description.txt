Wishify — приложение для создания и совместного использования списков желаний.

Создавайте вишлисты для дней рождения, свадеб, новогодних праздников и любых поводов. Добавляйте товары с ценами, ссылками и фотографиями. Делитесь с друзьями и родственниками одной ссылкой — без регистрации.

Друзья могут анонимно зарезервировать подарки или скинуться на что-то дорогое. Владелец не видит, кто что выбрал — сюрприз сохраняется.

КЛЮЧЕВЫЕ ВОЗМОЖНОСТИ:
• Неограниченное количество вишлистов
• Добавление товаров через ссылку — автоматически заполним название, цену и фото
• Шеринг по ссылке без регистрации для гостей
• Анонимное резервирование подарков — чтобы двое не купили одно
• Групповые покупки — копите на дорогой подарок вместе
• Обновления в реальном времени через WebSocket
• Offline-режим — просмотр без интернета

Wishify работает в паре с веб-версией на wishify.app — один аккаунт, везде.
